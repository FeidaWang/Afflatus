02

You review

You don't read every line. You ask three questions and make the AI answer them.

  1. Does it work?Run it. Click the thing.
  2. What breaks it?Empty input, huge input, double-click.
  3. Can I explain this diff?If not, ask the AI to explain it back — then decide.

The one law of this course: never merge code you can't explain.

03

You ship

Every session ends at a real URL, and the next session starts by reading a file — not by re-explaining yourself.

TODO.mdwrite the unfinished state before you close the chat
git commitsmall, often — cheap to throw away, cheap to redo
deployone command to a public URL, every time

Discard the conversation freely — the file is what persists.
