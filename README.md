# Project Afflatus

Deep-space captain log and personal investment blog. Current release: `v1.3`.

## 快速启动

```bash
cd "/path/to/Project Afflatus"
npm install
npm run dev
```

打开：

```text
http://127.0.0.1:5173/
```

## 构建

```bash
npm run build
```

## 生产预览

```bash
npm run preview
```

打开：

```text
http://127.0.0.1:4173/
```

完整技术说明见 [docs/TECHNICAL_GUIDE.md](docs/TECHNICAL_GUIDE.md)。
